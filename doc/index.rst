Django-Includes
===============

This package is a django application that enables "sub views" to be included executing both the "view" and "template" layers.

Supported ways:

* Direct include (same process will render the sub-view)
* Hinclude (an XMLHttpRequest will do the job)
* Edge side includes (a regular HTTP request will be issued)
